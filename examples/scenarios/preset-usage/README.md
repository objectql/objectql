# Preset Usage Example

This example demonstrates how to compose an application using Presets and Plugins.

It demonstrates:
1. **Preset**: Referencing `@example/basic-script` as a preset to inherit its defined objects
2. **Plugin**: Referencing `@example/audit-log` plugin to enable auditing capabilities
3. **App**: Composing them together in `objectql.config.ts`

## Run

```bash
pnpm install
pnpm run build
pnpm run serve
```
